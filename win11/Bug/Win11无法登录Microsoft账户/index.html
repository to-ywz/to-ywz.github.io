Win11无法登录Microsoft账户

首页 win11 Bug

昨天刷的 win11, Microsoft 登录账号失败了，琢磨半天才搞好，记个笔记

# 第一次遇到这个问题

记得当初 win10 上是直接在 clash 的 UWP Loopback 的管理器上配置了一下 Microsoft 的服务

然而，是我记性不好，这是 OneNote 无法同步的问题。整了半天都不好使

# WIN11 的解决方法

其实，可以先试试把代理关掉，我的就是这个 bug.
当然在这之前还尝试过修改代理，这里演示一下.
win + r 召唤 运行窗口

选择 网络和 Intent, 下面的子选项

进入后选择 更改网络适配器

选择当前在工作的适配器，右键属性


选择 IPv4, 双击

将 DNS 和 备用 DNS 改为 4.4.4.1 和 4.4.4.2

这样子就可以再试试能否登录成功，如果还是不行，只能说不是一种问题
记得登录完毕后将 DNS 修改回去
