# 本次涉及命令

# 数据库基础知识:

  1. 进入 MySQL 命令行界面 输入 show variables like '%dir%'; 可以 查看 数据库文件存放位置以及各个配置文件、插件存放位置

  2. 查看数据库
    在命令行 输入 show databases; 可以查看 本机 数据库目录下的数据库

  3. 创建数据库
    在命令行 输入 Create database 数据库名称; 可以创建数据库

  4. 创建数据库同时指定编码方式
    Create database 数据库名 default character set 编码格式;
    ubuntu 默认编码为 latin1 即 ISO-8859-1 支持中文存入，不过 不建议使用中文 作为数据库名或表名

    创建数据库时将 默认编码格式设置额为 uft8

  5. 使用选定数据库 use 数据库名;

  6. 修改数据库
    MySQL 数据库一旦安装成功，创建的数据库编码也就确定了，想要改变数据库的编码，可以使用
    Alter database 数据库名 default character set 字符集；
    不过很遗憾，命令不能修改 数据库名称

  7. 删除数据库
    数据库名字不能修改在，怎么办呢，那只能直接删除，重新建一个数据库了
    删除命令如下 drop database 数据库名称;

# 数据表的基本操作

# 修改数据表

  1. 修改表名
    命令格式: ALTER TABLE 旧表名 RENAME [TO] 新表名;

  2. 修改字段名
    命令格式: ALTER TABLE 表名 CHANGE 旧字段名 新字段名 新数据类型;

  3. 修改字段的数据类型
    命令格式: ALTER TABLE 表名 MODIFY 字段名 数据类型[约束条件];

  4. 添加字段
    命令格式: ALTER TABLE 表名 ADD 新字段名 数据类型 [约束条件][FIRST|AFTER 已存在字段名]

  5. 删除字段
    命令格式: ALTER TABLE 表名 DROP 字段名;

  6. 修改字段的排列位置
    命令格式: ALTER TABLE表名MODIFY字段名1数据类型FIRST|AFTER字段名2

  7. 删除数据表
    命令格式: DROP TABLE 表名;
