预防皮肤干燥

大道五十，天衍四十九，人遁其一！

# 起因

  1. 年龄大了
  2. 天气寒冷
  3. 偏食和减肥
  4. 经常使用洗衣粉、肥皂、洗洁精等洗涤剂及酒精等有机溶剂，以及长时间所处空调环境下等
  5. 血虚、阴虚的人皮肤更容易干燥
  6. 睡眠不足会使身体受到相当程度的伤害，血液循环也会变差

# 危害

长皱纹、暗黄、脱皮、起色斑，还易引发湿疹、特应性皮炎，
严重的时候肌肤甚至会像鱼鳞一样轻轻一抓就掉下很多白色皮屑

# 预防方案

# 饮食方面

多吃白色食物白萝卜、白菜、冬瓜、百合、银耳、莲藕、莲子这一类的都算，长得白的基本就是了，不知道白黄瓜算不算 (狗头)
除了这个法子还可以走药膳的路子:

# 马蹄甘蔗胡萝卜饮

食材:

  • 马蹄 100 克
  • 去皮甘蔗 250 克
  • 胡萝卜 1 根
  • 红枣 5 颗
  • 清水 1500 毫升

制作:

  • 马蹄去皮洗净
  • 甘蔗切成 10 厘米的长段后切成条
  • 红萝卜去皮切块
  • 红枣去核
    全部食材入锅，加水、冰糖，大火烧开转小火慢炖 1 小时即可

# 雪梨猪肺汤 (这个我觉得不太好喝)

食材:

  • 猪肺 500 克
  • 雪梨 250 克
  • 川贝母 20 克
  • 食盐适量

制作:

  • 雪梨洗净、连皮切成块状、去核
  • 川贝母洗净
  • 猪肺洗净、切成块状、沸水
    将适量清水放入煲内，煮沸后加入以上材料、猛火煲滚后改用慢火煲至食材熟烂、加盐调味即可

# 百合小米粥

食材:

  • 小米
  • 干百合
  • 干银耳
  • 红枣
  • 花生
  • 冰糖适量

制作:

  • 洗净食材
  • 往锅内注入清水大火煮沸，放入小米、银耳和花生，
  • 期间要注意搅拌以防糊底，等到材料加入完全煮沸后改为中火.
  • 在小米开始出花的时候将红枣、百合和冰糖放入小米粥中，将火力改为大火煮沸后即可出锅

# 茶

中华文化博大精深，衣食住行都是如此。养生当然少不了茶

# 玫瑰美颜茶

做法:

  • 干品玫瑰花 1g,
  • 麦冬、玉竹、首乌各 1.5g
  • 200 毫升的开水冲泡十分钟

# 雪梨茶

做法：雪梨切碎，冰糖煮梨水泡茶饮用
这个其实也可以止咳，个人咳嗽比较厉害，所以小时候有喝过这个
体质较寒的人可用乌龙茶 (这个不止咳)

# 补气生津茶 (这个还是挺好的)

  • 红枣 3 个
  • 西洋参 3 片
  • 麦冬 2g
  • 五味子 0.3g
  • 用 200 毫升的开水冲泡十分钟

# 日常防护

  1. 养成良好的作息习惯，多喝水
  2. 用温水洗澡，因为热水会将皮肤上的天然油分彻底洗掉
  3. 冬季洗澡一般不要超过 15 分钟
  4. 使用温和的沐浴液或香皂浴后涂上润肤品、身体乳、护手霜等
  5. 在那些易发生干裂的身体部位，涂润肤乳
  6. 可使用加湿器，增加空气湿度
